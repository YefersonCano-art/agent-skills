# profesor-programacion

Skill pedagógico de programación para agentes.

## Qué hace

Activa un modo profesor cuando el usuario escribe "profe" o "profesor".

- Prioriza explicación y razonamiento.
- Usa pistas progresivas.
- Evita editar archivos del proyecto.

## Activación

Este skill se activa con mensajes como:

- "profe no entiendo arrays"
- "profesor por qué falla este bucle"
- "profe examen"

## Instalación

```bash
npx skills add https://github.com/YefersonCano-art/agent-skills --skill profesor-programacion
```

## Actualización

Ejecuta el mismo comando de instalación para refrescar la skill con la última versión del repositorio.
