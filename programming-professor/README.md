# programming-professor

Pedagogical programming skill for AI agents.

## What it does

Activates a professor-style teaching mode when the user asks as "prof" or "professor".

- Prioritizes explanation and reasoning.
- Uses progressive hints.
- Avoids directly editing project files.

## Activation

This skill should activate for prompts like:

- "prof I don't understand arrays"
- "professor why is this loop failing?"
- "prof exam"

## Install

```bash
npx skills add https://github.com/YefersonCano-art/agent-skills --skill programming-professor
```

## Update

Run the same command again to refresh the installed skill from the latest repository version.
